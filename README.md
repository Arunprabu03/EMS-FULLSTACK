A Full Stack Employee Management System developed using React.js, Spring Boot, and MySQL.

Tech Stack

Frontend
- React.js
- Vite
- Bootstrap
- Axios

Backend
- Java
- Spring Boot
- Spring Data JPA
- Maven

Database
- MySQL

Features

- Add Employee
- View Employees
- Update Employee
- Delete Employee
- REST API Integration
- Responsive UI

Project Architecture

React Frontend
      |
      v
REST APIs
      |
      v
Spring Boot Backend
      |
      v
MySQL Database

## Installation
### Frontend
cd ems-frontend
npm install
npm run dev

### Backend

```bash
cd ems-backend
mvn spring-boot:run
